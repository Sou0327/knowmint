# KnowMint — Marketing Plan (WIP → v1)

作成日: 2026-02-21
ステータス: 調査を反映して肉付け中（最新版）

---

## 1. 目的 / ゴール

**目的**：4日後ローンチで「AIエージェントが自律購入できる市場」という新カテゴリを認知させる。

**60日KPI**
1. 自律購入デモの再生数
2. 初期5出品の購入率（5件中1件購入）
3. 出品者応募数（5〜10名）
4. MCP/CLI導入数

---

## 2. ターゲット（AIエージェント側 / 人間側）

### A) AIエージェント側（開発者 / 自動化勢）
- MCP/CLI/API を使う開発者
- “買った瞬間にタスクが進む”即効性を求める

### B) 人間側（出品者 / クリエイター）
- 経験やノウハウをマネタイズしたい人
- TikTok / X で発信経験がある層

---

## 3. 価値提案（ポジショニング / メッセージ）

**コアメッセージ**
- *The first marketplace where AI agents autonomously buy human knowledge.*

**補助メッセージ**
- *Sell real human expertise. Agents buy and use it instantly.*
- *Not a prompt library — a marketplace built for autonomous agents.*

---

## 4. 競合 / 代替手段（調査ベース）

### 既存の「プロンプト市場」
- **PromptBase**：プロンプト売買市場（$1.99–$9.99の価格帯）
- **FlowGPT**：コミュニティ中心の公開/共有型
- **AIPRM / PromptHero**：特定用途・ビジュアル/SEO特化

### 差別化軸
- 既存は「人間が探して買う」
- KnowMintは「**AIエージェントが自律購入して即実行**」

**参考ソース**
- PromptBase（マーケット型、単価帯）: https://promptbase.com/
- PromptBase含む比較: https://www.godofprompt.ai/blog/critical-review-popular-prompt-marketplace-platforms
- FlowGPTレビュー（コミュニティ型の特徴）: https://skywork.ai/blog/flowgpt-review-ai-prompt-marketplace/

---

## 5. チャネル戦略（無料最大化）

### A) AIエージェント向け
- **HN / Reddit / Dev.to / X / Discord**
- 形式: “デモ動画 + MCP導入スピード”
- 投稿軸: 「AIが自律購入して即実行」

### B) 人間向け
- **TikTok / X / Threads**
- 形式: “経験が売れる”の短尺動画
- 投稿軸: 「AIがあなたの知識を買ってくれる」

---

## 6. 4日ローンチ計画（詳細スケジュール）

### Day 1: 仕込み
- [ ] 自律購入デモ動画（30〜60秒）
- [ ] 初期5出品の作成
- [ ] 出品テンプレ整備
- [ ] LPトップメッセージ更新

### Day 2: クローズドβ
- [ ] AI開発者10人にテスト招待
- [ ] 購入フローの詰まり確認
- [ ] 3問アンケートでフィードバック

### Day 3: 公開準備
- [ ] MCP/CLI導入記事（短文でOK）
- [ ] “初期5出品”紹介記事

### Day 4: ローンチ
- [ ] X / HN / Reddit でデモ投稿
- [ ] TikTokで30秒デモ投稿
- [ ] 反応の良い投稿にスレッド追加

---

## 7. コンテンツ設計

1. **自律購入デモ動画**（30〜60秒）
2. **MCP/CLI導入記事**（5分セットアップ）
3. **初期5出品紹介記事**

---

## 8. 投稿テンプレ / コピー集

### AI向け（X/Reddit/HN）
```
AI agents can now buy human knowledge.
We built a marketplace where agents purchase and execute instantly.
Demo: <URL>
MCP/CLI ready.
```

### 人間向け（X/TikTok）
```
あなたの経験が“売れる知識”になる。
AIが直接買ってくれるマーケットを作った。
出品はテンプレに沿って書くだけ。
```

---

## 9. デモ台本（短尺動画）

1. 検索 → 該当知識を表示
2. 自律購入ボタン
3. 即タスク実行（CLI/MCP）
4. 結果表示

---

## 10. KPI / 計測

1. デモ動画再生数
2. 初期出品購入率
3. 出品者応募数
4. MCP/CLI導入数

---

## 11. 運用フロー（誰でも回せる手順）

2. 週2回：X投稿（AI向け/人向け）
